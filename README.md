## 3. 구현할 기능

### - 패스워드 생성기
1. 슬라이더로 조절하는 생성 문자열 길이 (최소 8 ~ 최대 120)
1. 숫자/영소문자/영대문자/특수문자 포함 여부 체크박스 혹은 토글 스위치
1. 사용 체크되어있는 문자는 무조건 1개이상 포함
1. 사용할 수 있는 특수문자 체크박스 혹은 토글 스위치
1. 자주 쓰는 특수문자 모음 프리셋(예; {!@#$%^}세트 등) 제공
1. 특수문자 모두 선택/선택 해제 버튼
1. 클립보드로 복사 버튼
1. 복사 시 토스트 오버레이 메시지 혹은 복사 버튼이 초록색 체크 표시로 변환
1. 패스워드 재생성 버튼
1. 생성 결과 화면은 실시간으로 변동하며, 영문은 검은 글씨, 숫자는 파란 글씨, 특문은 빨간 글씨로 표시

### - 문자열 암/복호화 페이지
1. 문자열을 받아 AES-128 CBC 모드로 암호화 / 복호화
1. 알고리즘은 AES-128, 운영 모드는 CBC, 패딩 방식은 PKCS7, 인코딩은 평문 UTF-8, 암호문 출력은 BASE64
1. 텍스트 파일 업로드/다운로드를 지원. 파일 확장자는 .aes
